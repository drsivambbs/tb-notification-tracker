# Screens

This directory contains screen/page widgets for the application.

## Planned Screens:
- `login_screen.dart` - Login page
- `dashboard_screen.dart` - Dashboard with metrics
- `case_entry_screen.dart` - Case entry form (PHC users)
- `case_list_screen.dart` - Case list with filtering
- `users_screen.dart` - User management (Admin only)
- `app_scaffold.dart` - Main layout with sidebar navigation
